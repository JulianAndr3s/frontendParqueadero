Bienvenido

Este aplicativo emula el sistema de un parqueadero.

Ingresa al apartado de Vehículos en la barra de navegación para iniciar las funcionalidades.


Esta aplicación Web, tiene las opciones de:


Ingresar

Mediante esta opción se puede ingresar un vehículo con sus datos pertinentes a la base de datos

Actualizar

Mediante esta opción se puede actualizar los datos de un vehículo ya ingresado.

Buscar

Mediante esta opción se puede buscar un vehículo mediante su placa correspondiente.

Eliminar

Mediante esta opción se puede eliminar un vehículo, una vez haya salido del parqueadero.
